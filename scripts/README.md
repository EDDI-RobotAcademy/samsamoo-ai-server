# Development Scripts

This directory contains development, testing, and utility scripts for SamSamOO-AI-Server.

## Scripts

### Demo Scripts
- **demo_llm_providers.py** - Standalone demonstration of multi-provider LLM system
- **quick_demo.py** - Quick demonstration script

### Testing Scripts
- **test_llm_providers.py** - LLM provider testing utilities
- **test_pdf_generation.py** - PDF generation testing

### Utility Scripts
- **fix_fontconfig_warning.py** - One-time fix for Fontconfig warnings on Windows

## Usage

Run scripts from the project root directory:

```bash
cd SamSamOO-AI-Server
python scripts/demo_llm_providers.py
```

## Note

These scripts are for development and testing purposes only and are not part of the production application.
