## Report Formats

The system generates multiple report formats:

### PDF Report
- Professional formatting with ReportLab
- Embedded charts and tables
- 8-10 pages with executive summary

### HTML Report
- Interactive web-based format
- Responsive design
- Print-friendly styling

### Charts
Generated using matplotlib:
- Ratio bar charts
- Trend analysis
- Performance indicators
